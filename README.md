  ## Description
  
  A web application that will use a backend using express.js to store and update the static page website.
  
  <p align="right">(<a href="#readme-top">back to top</a>)</p>
  
  
  ## Installation
  
  1. Open intergrated terminal and run "npm i" 
  2. Type "node server.js" into the intergrated terminal so that the website is hosted onto a local host port. 
  3. Click on link provided in the console to open the website in the browser.
  
  <p align="right">(<a href="#readme-top">back to top</a>)</p>
  
  ## Usage
  
  VSCODE or similar application is nessary to host the server
